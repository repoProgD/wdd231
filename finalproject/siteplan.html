Website Plan

Site Name: Visit AR

The name "Visit AR is clear, concise and directly conveys the purpose of the site. The use of "Visit + country" is a common convention for tourism websites in English, making it easily recognizable and easy to remember. In addition, in Spanish "visitar" means "to visit", which is a play on words that reinforces the site's purpose.

Site Purpose

The purpose of this site is to help users discover and explore travel destinations in Argentina. The site provides information about various locations. It also allow users to comare options, and offers curated recommendations to guide their decision-making

Scenarios

  • Which destinations there are available in Argentina?
  • How these destinations compare to each other?

Color Scheme

The color scheme for this project is represented by the following HEX codes:

  • Primary Color: #6CACE4 - Use for header and footer background
  • Secondary Color: #E7EEFB - Use for main background
  • Overlay Color: #1512F5 - Use for cards and other UI elements
  • Accent1 Color: #FF8E18 - Use for primary interactive highlights, borders, and underlines
  • Accent2 Color: #E1D600 - Use for secondary interactive highlights, borders, and underlines
  • Text Color (light): #FEFEFE - Use for text on dark backgrounds
  • Text Color (dark): #000000 - Use for text on light backgrounds
  • text-dark2-color: #020640 - Secondary use for text on light backgrounds

Typography

  • Heading Font: Lora - Use for all headings
  • Body Font: Roboto - Use for all body text
